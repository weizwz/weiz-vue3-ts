<h1>vue3+ts 常用界面模板</h1>

## 介绍

框架基于 [Pure-Admin-Thin 精简版](https://pure-admin.github.io/pure-admin-doc/pages/introduction/)。

技术：vite5 + vue3 + ts + pina + Element-Plus + Tailwindcss

## 提交规范

- feat: 增加新功能
- fix: 修复问题/BUG
- style: 代码风格相关无影响运行结果的
- perf: 优化/性能提升
- refactor: 重构
- revert: 撤销修改
- test: 测试相关
- docs: 文档/注释
- chore: 依赖更新/脚手架配置修改等
- workflow: 工作流改进
- ci: 持续集成
- types: 类型定义文件更改
- wip: 开发中
